CASE STUDY FOR MEET APP

Overview

Meet App was a personal project I built as part of my Full-Stack Web Development course at CareerFoundary. This Meet App goal was to create an event site where users can easily view events. Build a lightweight event management platform by integrating Google Calendar API with a serverless (FaaS) backend.

Technology Stack

Frontend React, Create React App (CRA), CSS, Axios, Recharts, Jest (Enzyme, Cucumber, Puppeteer) → (Frontend Test) Atatus (performance monitoring), Github-pages → (Frontend Hosting) Github
Backend Serverless, AWS Lambda, OAuth (Google Cloud Console) Github

Achievements & Lessons Learned

Achievements

  • Developed a full-stack, serverless event discovery platform
    • → Built both frontend and backend independently, using AWS Lambda and React.
  • Integrated real-time data from Google Calendar API
    • → Successfully fetched, displayed, and managed live event data.
  • Implemented Progressive Web App (PWA) capabilities
    • → Enabled offline access and installable app experience using service workers.
  • Built interactive data visualizations
    • → Created dynamic charts that allow users to quickly understand event trends by city and genre.
  • Achieved 90%+ test coverage
    • → Used Jest and Cucumber to ensure high reliability through rigorous testing. li>Mastered OAuth2 Authentication Flow → Implemented secure login and access management using industry standards.
  • Deployed globally on Vercel for optimal performance
    • → Achieved fast load times and minimal downtime with serverless deployment.
  • Enhanced user engagement through responsive design
    • → Optimized UX for both desktop and mobile users, improving accessibility and retention.

Lessons Learned

  • Serverless architecture simplifies deployment and scaling but requires careful management of cold starts and token handling.
  • Working with third-party APIs like Google Calendar requires robust error handling and quota management.
  • Visualizing data through charts significantly enhances user engagement and provides deeper insights.
